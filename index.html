Responsiva

Open Source Responsive CSS Framework

Download .zip Download .tar.gz View on GitHub

Simple. Lightweight. Beautiful


Examples:

  • Responsiva - Simple Grid system
  • Responsiva - Stuck Grid system


Features

  • lightweight
  • simple & clear
  • 12 columns layout
  • reset.css included
  • helper classes
  • rtl support (responsivo.rtl.css [under development])

Usage

1) include Responsiva in your HTML template file

2) Basic layout example:

<div class="main_container">
<div class="row">
<div class="col-6"></div>
<div class="col-6"></div>
</div>
<div class="row">
<div class="col-8"></div>
<div class="col-4"></div>
</div>
</div>
